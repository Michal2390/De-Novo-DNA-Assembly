# Asemblacja-de-novo-DNA

Repozytorium zawiera materialy do laboratorium MBI dotyczacego asemblacji de novo DNA.

## Zawartosc
- `lab1/` - komplet plikow do Lab 1:
  - skrypt GC-content (`lab1/scripts/gc_content.py`)
  - test skryptu (`lab1/tests/test_gc_content.py`)
  - raport roboczy (`lab1/sprawozdanie_lab1_draft.md`)
  - wyniki QUAST (`lab1/output/quast_results/`)

## Szybki start
Szczegoly uruchamiania i parametry dla Lab 1 sa w:
- `lab1/README.md`

## Uwaga
W repo trzymane sa lekkie artefakty (kod i raporty tekstowe).
Duze pliki wynikowe sa celowo pomijane przez `.gitignore`.
